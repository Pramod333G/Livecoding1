What is Java?


Java is an object-oriented programming language that was developed by James Gosling at Sun Microsystems,which is now acquired by Oracle,In today's time ,Java is one of the most diverse languages. Java programming language is a high-level object oriented language designed to have as few implementation dependencies as possible


Key features of Java Programming language

Here are the important features of the java language

  1. Java programming language is designed to built object-oriented applications
  2. Java is a multithreaded language with automatic memory management
  3. In java, there is a feature called WORA i.e write once run anywhere after complition
  4. Java code can be run on any computing platform
